## Technical Details
### Technologies/Components Used
For Software:
- [GP2040-CE firmware](https://gp2040-ce.info/)

For Hardware:
- raspberry pi pico 
- limit switches 
- tactile button 


# Installation
- download the GP2040.uf2 file
- connect the pico to your pc while pressing the boot/selc button
- paste the GP2040.uf2 file in rpdrive
